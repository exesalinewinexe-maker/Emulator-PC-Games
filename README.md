# Emulator-PC-Games
A PC-based game emulator and compatibility layer written from scratch. Designed to load, analyze, and simulate PC game behavior, with future plans for physics tools, modding utilities, and experimental emulation features. This project includes support games that are: BeamNG.drive physics crash game, Forza Horizon 5 (FH5), and Forza Horizon 6 (FH6).

# 🔮 Future Plans
The Emulator-PC-Games project is designed to grow in a full PC game compatibility layer with advanced tools for analysis, modding, and physics simulation. Planned future features include:

• Advanced API simulation for modern PC games
