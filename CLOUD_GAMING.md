# Cloud Gaming Guide

Cloud gaming lets you play PC games on ANY device without installing them.

## Services
- Xbox Cloud Gaming
- NVIDIA GeForce NOW
- Amazon Luna
- Steam Link (local streaming)
- Moonlight (NVIDIA GPU streaming)
- Sunshine (open-source streaming host)

## Best Use Cases
- Low-end PCs
- Phones and tablets
- Chromebooks
- Smart TVs

## Recommended Games
- Fortnite
- Cyberpunk 2077
- Halo Infinite
- GTA V
